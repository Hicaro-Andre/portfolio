## Sobre

Portfólio desenvolvido para centralizar minha trajetória como desenvolvedor Full Stack — projetos reais, experiências e formação em um só lugar. Interface em PT/EN com animações e formulário de contato integrado.

---

## Stack

- React 19 + TypeScript
- Vite
- React Router DOM
- Framer Motion + tsParticles
- EmailJS

---

## Como rodar

```bash
npm install
npm run dev
```
